# Bank Loan Prediction using Logistic Regression

## Project Overview
This project focuses on predicting whether a bank customer will accept a personal loan offer using a Logistic Regression model.

The objective is to understand the complete machine learning workflow including data preprocessing, feature engineering, model training, and performance evaluation.

## Workflow
- Data Loading and Exploration
- Data Cleaning and Feature Engineering
- Train-Test Split
- Feature Scaling
- Logistic Regression Model Training
- Model Evaluation using Accuracy and ROC-AUC

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Dataset
The dataset contains customer demographic and financial information used to predict personal loan acceptance.

## Repository Structure
- `Bank_Loan_Logistic_Regression.ipynb` – Complete machine learning workflow
- `Bank_Personal_Loan_Modelling.csv` – Dataset used for training and testing
